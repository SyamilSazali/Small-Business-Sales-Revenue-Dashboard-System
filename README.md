# Small Business Sales & Revenue Dashboard System

A web-based sales and revenue dashboard system for small businesses built using PHP, MySQL, and XAMPP that allows product and sales management, automatic revenue and profit calculation, date-based filtering, and PDF report generation for business analysis and record keeping.

## Features
- Manage product data (add, update, delete)
- Manage sales records
- Automatic calculation of total sales and profit
- Filter sales records by date
- Generate printable sales reports (PDF)

## Tech Stack
- PHP
- MySQL
- XAMPP
- HTML, CSS, JavaScript
- Dompdf (PDF generation)

## Project Purpose
This project was developed to simulate a real-world small business system and improve understanding of web development, database management, and reporting systems.

## Database
- Uses MySQL database to store all system data
- Database must be created and imported before running the system
- Users can generate dummy data using AI tools for testing purposes

## How to Run

### 1. Install XAMPP
- Download and install XAMPP
- Start Apache and MySQL

### 2. Setup Database
- Go to http://localhost/phpmyadmin
- Create database (e.g. sales_db)
- Import .sql file

### 3. Setup Project
- Copy project folder into:
  C:\xampp\htdocs\

### 4. Run System
- Open browser
- Go to:
  http://localhost/small-business-dashboard

## Important Notes
- Extract/unzip Dompdf folder before running
- Ensure XAMPP is running
- Import database first or system will not work
